# AI Career Copilot 🚀

AI-powered platform that helps students prepare for placements using resume analysis, AI interviews, coding practice, and career roadmap generation.

## Features

✔ Resume Analyzer  
✔ ATS Resume Score  
✔ Job Description Analyzer  
✔ Skill Gap Detection  
✔ Career Roadmap Generator  
✔ AI Mock Interview  
✔ Voice Mock Interview  
✔ Coding Question Generator  
✔ Admin Dashboard  
✔ Docker Containerization  
✔ CI/CD Pipeline  

## Tech Stack

Frontend: Streamlit  
Backend: FastAPI  
Database: MongoDB Atlas  
AI APIs: GrokAI 
DevOps: Docker + GitHub Actions  

## Architecture

User → Streamlit → FastAPI → AI APIs → MongoDB
